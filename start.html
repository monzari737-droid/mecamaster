Meca Master

Application de mise en relation automobile

⚠️ IMPORTANT Cette application utilise Next.js (React), PAS de fichiers HTML statiques. Les pages sont générées dynamiquement par Node.js.
1

Autoriser PowerShell (Windows)

Ouvrez PowerShell en administrateur et exécutez :

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
2

Aller dans le dossier

cd "C:\Users\user\Desktop\MECA MASTER\meca-master"
3

Installer les dépendances

npm install
4

Lancer l'application

npm run dev
